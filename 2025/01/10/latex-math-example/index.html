LaTeX数学公式示例

Created2025-01-10|Updated2025-07-06
|Post Views:

这篇文章演示了如何在Hexo博客中使用LaTeX数学公式。

行内公式

这是一个简单的行内公式：$E = mc^2$，爱因斯坦的质能方程。

另一个例子：当 $a \ne 0$ 时，一元二次方程 $ax^2 + bx + c = 0$ 的解为 $x = \frac{-b \pm \sqrt{b^2-4ac}}{2a}$。

块级公式

积分公式

$$\int_{a}^{b} f(x)dx = F(b) - F(a)$$

求和公式

$$\sum_{i=1}^{n} i = \frac{n(n+1)}{2}$$

矩阵

$$\begin{pmatrix}
a & b \
c & d
\end{pmatrix}
\begin{pmatrix}
x \
y
\end{pmatrix}

\begin{pmatrix}
ax + by \
cx + dy
\end{pmatrix}$$

分数和根号

$$\frac{d}{dx}\left( \int_{0}^{x} f(u),du\right)=f(x)$$

$$\sqrt{x^2 + y^2} = r$$

希腊字母

$$\alpha + \beta = \gamma$$
$$\sum_{i=1}^{\infty} \frac{1}{i^2} = \frac{\pi^2}{6}$$

常用LaTeX语法

  • 上标：x^2 → $x^2$
  • 下标：x_1 → $x_1$
  • 分数：\frac{a}{b} → $\frac{a}{b}$
  • 根号：\sqrt{x} → $\sqrt{x}$
  • 求和：\sum_{i=1}^{n} → $\sum_{i=1}^{n}$
  • 积分：\int_{a}^{b} → $\int_{a}^{b}$
  • 希腊字母：\alpha, \beta, \gamma → $\alpha, \beta, \gamma$

现在你可以在Hexo博客中自由使用LaTeX数学公式了！
